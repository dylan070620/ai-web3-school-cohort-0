# Foundry Experiments

Use this folder for Solidity, tests, scripts, and deployment notes.

## First Goals

- Install or verify Foundry.
- Create a minimal project.
- Write or reuse a simple contract.
- Run local tests.
- Learn deployment scripts.
- Deploy to a testnet only after private key safety is clear.

## Deployment Safety Checklist

- [ ] Use a test wallet only.
- [ ] Keep private key in `.env`.
- [ ] Never commit `.env`.
- [ ] Use a testnet RPC URL.
- [ ] Confirm contract and network before broadcasting.
- [ ] Save transaction hash only if safe to publish.
