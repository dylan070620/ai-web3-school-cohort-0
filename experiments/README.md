# Experiments

Use this folder for hands-on experiments.

Suggested experiment tracks:

- `python-api/`: Python scripts, API calls, JSON parsing.
- `llm-api/`: first LLM API calls, prompts, structured outputs.
- `foundry/`: Solidity tests, scripts, deployments.
- `onchain-data/`: wallet, transaction, token, and contract analysis.
- `agent/`: tool-using Agent experiments.

Keep secrets in local `.env` files only. Commit `.env.example`, not `.env`.
