# On-Chain Data Experiments

Use this folder for wallet, transaction, token, and contract analysis.

## First Questions

- What transactions did this wallet send?
- Which contracts did it interact with?
- What token transfers happened?
- Can an AI model summarize the behavior in plain language?

## Possible Data Sources

- Public RPC endpoint.
- Block explorer API.
- Dune or similar analytics platforms.
- Downloaded CSV files.

## Output Style

Keep outputs beginner-readable:

- Short summary.
- Table of important events.
- Risks or unknowns.
- Links to public-safe proof.
