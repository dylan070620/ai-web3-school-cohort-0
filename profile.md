Web3:

Programming:

- I know some Python and Solidity.

## Time Budget

Daily available time: 3 to 4 hours.

Recommended daily split:

- 45 minutes: concepts and Handbook reading.
- 75 minutes: hands-on coding or deployment.
- 45 minutes: data or Agent experiment.
- 30 minutes: daily log, task proof, and reflection.
- 15 minutes: cleanup, questions, and next action.

## Main Direction

On-chain data analysis plus AI Agent.

Target capability:

- Use Python to call APIs and process data.
- Use LLM APIs to summarize, classify, and reason over blockchain data.
- Use Foundry to test and deploy simple contracts to a testnet.
- Build a small Agent that can inspect on-chain activity and generate useful analysis.

## Public Learning Rules

- Prefer small daily actions over big vague plans.
- Record proof of work as links, screenshots, commands, or short notes.
- Keep sensitive information out of the repo.
- Ask AI for help, but write down what I actually understand.
- When blocked, create a small reproducible note instead of waiting.
