# Python API Experiments

Start here before building Agents.

## First Experiment

Write a Python script that:

1. Sends a GET request to a public API.
2. Reads the status code.
3. Parses JSON.
4. Prints only the fields you need.

## Concepts

- HTTP request.
- HTTP response.
- Status code.
- JSON.
- Headers.
- API key.
- Environment variables.
